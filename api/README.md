# Backend API

This is the backend REST API built with Express.js and MySQL.

## Prerequisites
- Node.js installed
- MySQL server running

## How to Start

1. Go to the `api` directory.
2. Configure your MySQL credentials in the `.env` file.
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Run the server:
   ```bash
   npm start
   ```

## Tech Stack
- Node.js
- Express.js
- MySQL
